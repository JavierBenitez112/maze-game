Un juego de laberinto 3D desarrollado en Rust utilizando la librería Raylib.

## Descripción

Maze Game es un proyecto que implementa un juego de exploración de laberintos con gráficos 3D. El juego cuenta con un sistema de menú, movimiento del jugador, casting de rayos (raycasting) para renderizado 3D y diversos efectos visuales.

## Tecnología

- **Lenguaje**: Rust
- **Gráficos**: Raylib
- **Tipo de proyecto**: Juego 3D

## Características

- Exploración de laberintos generados
- Renderizado 3D mediante raycasting
- Sistema de menú interactivo
- Movimiento del jugador en tiempo real
- Efectos visuales y texturas
